กราฟที่ควรทำ

  1. Bar Chart — CodeQL Security Alerts by Strategy
  - แกน X: Strategy (BP, CE, CS, SD)
  - แกน Y: จำนวน Alerts
  - แยกสี: High vs Medium
  - จุดเด่น: เห็นความแตกต่างชัดเจน (CE=11, BP=5, CS=0, SD=0)

  2. Stacked Bar Chart — CodeQL Alerts by Version (แยกตาม Feature)
  - แกน X: แต่ละ version (IMBP01, IMCE01, SCCE01, PDCE01, ...)
  - แกน Y: จำนวน Alerts
  - Group by Feature (IM, SC, PD)
  - จุดเด่น: เห็นว่า version ไหนมีปัญหาเยอะที่สุด

  3. Grouped Bar Chart — DAST Warnings by Strategy
  - แกน X: Strategy
  - แกน Y: Avg Warnings / Avg Pass
  - จุดเด่น: เปรียบเทียบ security posture ของแต่ละ strategy

  4. Heatmap — DAST Warnings by Version
  - แกน X: Version (12 ตัว)
  - แกน Y: Warning types (7 ชนิด)
  - สี: มี/ไม่มี warning
  - จุดเด่น: เห็นภาพรวม Server Leak ที่มีแค่ 4 versions

  5. Radar Chart (Spider Chart) — Overall Strategy Comparison
  - 3 แกน: Test Pass Rate, CodeQL Score (inverse), DAST Score
  - 4 เส้น: BP, CE, CS, SD
  - จุดเด่น: เปรียบเทียบภาพรวมทุกมิติในกราฟเดียว

  6. Summary Table Chart — Unit Tests Pass Rate by Strategy x Feature
  - ตาราง heatmap: Strategy (rows) x Feature (columns)
  - ค่า: pass rate (ทุกตัว 100%)
  - จุดเด่น: ยืนยันว่า functional correctness ผ่านหมดทุก strategy
